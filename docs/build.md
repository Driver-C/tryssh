# Build Project

## Prerequisites

- Go 1.25 or later
- golangci-lint (for linting)

## Build binary file for the current architecture

```bash
make
```

## Run tests

```bash
make test
```

## Run linter

```bash
make lint
```

## Generate coverage report

```bash
make coverage
```

## Cross-compilation

```bash
make multi
```

## Clean binary packages

```bash
make clean
```
